# Dev Guides

Bem-vindo(a) ao **Dev Guides**! Este repositório é um porto seguro para programadores iniciantes (e para aqueles momentos em que a nossa memória falha). Aqui você encontra guias práticos, direto ao ponto e estruturados em etapas para configurar projetos, entender arquiteturas e dominar ferramentas de desenvolvimento.

A ideia é simplificar o "como começar" para que você foque no que realmente importa: **construir**.

---

## O que você vai encontrar por aqui?

Os tutoriais são divididos por ecossistemas e tecnologias, mapeando desde a configuração do ambiente virtual até a estrutura de pastas do projeto.

```text
📂 python/
└── 📄 django-tutorial.md  # Setup de Django, apps, URLs e Layout Base (Mestre)
└── 📄 python-libs.md      # Guia de bibliotecas essenciais
📂 ruby/
└── 📄 rails-setup.md     # Criação de apps Rails com PostgreSQL, MVC e Partials
```

---

## Tecnologias Cobertas (por enquanto!)

- **Python:** Ecossistema Django, gerenciamento de ambientes virtuais (`venv`) e arquitetura MVT.
- **Ruby:** Framework Web Ruby on Rails, gerenciamento de banco de dados com PostgreSQL e reaproveitamento de views.
- **Boas Práticas:** Padrões de desenvolvimento, organização de arquivos estáticos globais e comandos de console essenciais.

---

## Como usar este repositório?

1. **Navegue pelas pastas:** Entre na pasta da linguagem ou tecnologia que você está estudando no momento.
2. **Siga o passo a passo:** Os guias possuem uma **Ordem de Desenvolvimento Sugerida** para garantir que você não pule nenhuma etapa crítica (como migrações de banco de dados).
3. **Copie e adapte:** Use os blocos de código prontos como um *boilerplate* (molde inicial) para os seus próprios projetos acadêmicos ou pessoais.

---

## Contribuições

Aprendeu um truque novo, configurou um framework diferente ou achou um jeito mais fácil de explicar algum passo? Contribuições são super bem-vindas!

1. Faça um **Fork** do projeto
2. Crie uma branch para sua modificação (`git checkout -b feature/novo-guia`)
3. Abra um **Pull Request** explicando o que foi adicionado
